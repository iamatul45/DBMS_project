# How To
* Install Django
```
pip install django
```
* Create Superuser
```
py manage.py createsuperuser
```
* migrations
```
py manage.py makemigrations
py manage.py migrate
```
* runserver
```
py manage.py runserver
```
and go to link http://127.0.0.1:8000/
